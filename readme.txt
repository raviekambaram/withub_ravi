The main aim is to build a hub for Women in Technology online, 
where members can do the following : Sign up / log in as a member 
Update contact information / status (student, alumni, etc) 
View / post / review / comment on events Request a mentor / mentee, 
Also a job search module was built where all the members of the site 
would be able to find new jobs and apply for one and also allows 
employers to post new jobs.
